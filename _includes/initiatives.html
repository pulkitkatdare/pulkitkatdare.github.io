Statistical v/s Neural Machine Translation

Course Project @IIT Bombay, India
July 2016 to December 2016

Trained the Neural Machine Translation model with embedded attention mechanism for transliteration. Compared the alignments probabilities computed using Statistical Machine Translation (SMT) and Neural Machine Translation.

Character Recognition using Unsupervised SNNs

IIT Bombay, India/address>January 2015 to April 2015

Designed an unsupervised learning algorithm along with lateral inhibition using spiking neural networks(SNNs) for digit recognition on MNIST database. The database achieved an accuracy of 37%, which could be further improved by using genetic algorithms

Using Vanishing Points for Camera Calibration

IIT Bombay, India/address>January 2015 to April 2015

Calibrated a standard camera using the geometric concepts behind mutually perpendicular vanishing points. Achieved accuracy turned out to be low(0.001%) when compared with a calibrated camera
